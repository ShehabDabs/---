# موقع صُنّاع الأمل - خطة التطوير

## إرشادات التصميم

### مراجع التصميم
- **النمط**: تصميم عربي حديث مع عناصر متحركة تعبر عن ذوي الهمم
- **الألوان الأساسية**: 
  - الأساسي: #2E7D32 (أخضر الأمل)
  - الثانوي: #1565C0 (أزرق الثقة)
  - المساعد: #FF6F00 (برتقالي التفاؤل)
  - النص: #212121 (رمادي داكن)
  - الخلفية: #FAFAFA (أبيض مكسور)

### الخطوط
- العناوين: Cairo font-weight 700
- النصوس العادية: Cairo font-weight 400
- النصوص المهمة: Cairo font-weight 600

### العناصر المتحركة (بدون صور)
- أيقونات CSS متحركة لذوي الهمم (كرسي متحرك، عصا بيضاء، لغة الإشارة)
- رسوم متحركة بـ CSS تعبر عن الدعم والتضامن
- تأثيرات حركية للنصوص والعناصر

---

## ملفات المشروع

### 1. index.html
- الصفحة الرئيسية مع قسم ترحيبي
- شريط التنقل العربي
- عناصر متحركة تعبر عن ذوي الهمم
- أقسام: الرئيسية، عن المبادرة، الخدمات، الأنشطة، التطوع، التواصل

### 2. style.css
- تصميم متجاوب باللغة العربية (RTL)
- رسوم متحركة بـ CSS فقط
- أيقونات وعناصر متحركة لذوي الهمم
- تأثيرات بصرية جذابة

### 3. script.js
- تفاعلات JavaScript للتنقل السلس
- تحكم في الرسوم المتحركة
- تأثيرات التمرير

## المهام

1. **إعداد الهيكل الأساسي** - تحديث HTML الأساسي
2. **تصميم الصفحة الرئيسية** - قسم ترحيبي مع عناصر متحركة
3. **إنشاء أقسام المحتوى** - جميع الأقسام المطلوبة
4. **تطبيق التصميم** - CSS كامل مع الرسوم المتحركة
5. **إضافة التفاعلية** - JavaScript للتنقل والتأثيرات
6. **اختبار التجاوب** - التأكد من عمل الموقع على جميع الأجهزة
